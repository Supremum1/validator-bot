# Валидатор заявок в Telegram-группу

Telegram-бот для проверки заявок на вступление в чат студентов. Пользователь отправляет фото студенческого билета, модератор проверяет его и принимает или отклоняет заявку через inline-кнопки.

## Как работает

1. Пользователь отправляет заявку на вступление в закрытую Telegram-группу.
2. Бот получает событие `ChatJoinRequest`.
3. Бот пишет пользователю в личные сообщения и просит отправить фото студенческого билета.
4. Пользователь отправляет фото.
5. Бот пересылает фото в чат модераторов.
6. Под фото появляются кнопки `Принять` и `Отклонить`.
7. Если модератор нажимает `Принять`, бот одобряет заявку пользователя.
8. Если модератор нажимает `Отклонить`, бот отклоняет заявку.
9. Пользователь получает сообщение с результатом проверки.
